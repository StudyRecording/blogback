# 个人博客
简单的个人博客实现（个人学习练习使用，原本想搭建一个，但是花钱😂）
## 实现框架
Spring Boot + Jpa + Thymeleaf  
使用MySql数据库  
前端UI组件样式使用Semantic UI  
使用的缓存工具为Ehcache

 
## 功能
- 博客前台
    - 博客显示
    - 分类
    - 标签
    - 查询
    - 关于我
    - 评论
- 博主平台
    - 登录
    - 添加、修改博客
    - 添加、修改标签
    - 评论的查看与回复
    - 管理员个人信息的修改
