# 个人博客
简单的个人博客实现（个人学习练习使用，原本想搭建一个，但是花钱😂）
## 实现框架
Spring Boot + Jpa + Thymeleaf  
使用MySql数据库  
前端UI组件样式使用Segment UI
 
## 功能
- 博客前台
    - 博客显示
    - 分类
    - 标签
    - 查询
    - 关于我
    - 评论
- 博客后台
    - 登录
    - 添加、修改博客
    - 添加、修改标签
